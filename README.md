Album-Downloader
================
Some simple image spider examples using scrapy.

## Requirements

使用包管理器pip / easy_install

依赖于scrapy, PIL:

`pip install scrapy`

`pip install PIL`

## Usage

`scrapy crawl douban_image_spider`
